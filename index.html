About Sampage

Samepage Health is a low-risk, evidence-driven turnkey solution for providers to effectively take-risk on chronically ill patients– changing patient behaviors that improve measurable outcomes while reducing costs. Samepage brings each organization technology, workflows, program management, clinician training, and ongoing oversight, while simultaneously ensuring each deployment is fully controlled by the sponsoring provider

Modular, Integrated Offerings

Core components of Samepage's Care Transformation Program are:
  • Goal driven care model transformation centered around the TEAMCare™ program developed at the University of Washington
  • The Change Path patient/provider collaboration SaaS-software for the standardized delivery of the behavior change program
  • Ongoing program management & continual improvement, including systemic case review and clinician E-Learning environment

Each of the above can operate stand-alone, or in combination
